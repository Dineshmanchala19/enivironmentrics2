About Our Project

Our Purpose

Our project, AquaSoil, aims to leverage remote sensing technology to analyze and monitor soil moisture levels and vegetation health. By using NDVI, NDWI, and NSMI indices, we provide valuable insights for farmers, researchers, and environmentalists.

Technology Stack

  • Flask - A lightweight WSGI web application framework in Python
  • Rasterio - For reading and writing geospatial raster data
  • Jinja2 - Templating engine for rendering HTML
  • Tailwind CSS - Utility-first CSS framework for styling
  • PostgreSQL - Relational database management system (if applicable)

Meet the Team

We are a passionate team of developers and researchers dedicated to enhancing agricultural productivity through technology.
